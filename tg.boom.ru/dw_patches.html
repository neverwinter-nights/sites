Transport Giant v1.2 -> 1.3 patch - 30 Октября 2004

Патч для игры "Транспортный Олигарх" (Transport Giant) версии 1.3 (1.30)
Английская версия, 3.74 Мб.

В патч 1.3 (1.30) были добавлены следующие особенности и исправления:

  • Shift-клик на транспортном средстве активирует расписание для всех транспортных средств с таким же расписанием. (таким образом можно менять несколько транспортных средств, имеющих абсолютно идентичные расписания или менять их место назначения и т.д. - чтобы произошёл обмен, нужно иметь достаточное количество транспортных средств в депо, в противном случае обмен не произойдёт).

  • Имя может быть изменено с помощью Shift-клика на терминале (изменения не действуют на уже существующих линиях).
  • Сильно улучшились взлёт и посадка самолётов (у садящихся самолётов приоритет над взлетающими; в аэропорте могут двигаться до трёх самолётов одновременно).
  • Новые промышленности теперь появляются выборочно (более частое появление современных промышленностей в бесконечной игре).
  • Теперь на концах дороги может быть положен буффер.
  • Различные другие исправления.

    Нормальная работа старых сохранённых игр не гарантируется после установки патча 1.3 (1.30).

  • Выход из игры из-за соперника -> исправлено
  • Возможный выход из игры когда два поезда встречаются в паралельных дугах -> исправлено
  • Компьютерный противник всегда использовал одни и те же имена -> исправлено

  • Выход из игры из-за того, что пересечение дорог было на конце пути -> исправлено

  • Выход из игры при расширении платформы (расписание взорвавшегося транспортного средства, удаление станции в расписании и расширение другой) -> исправлено
